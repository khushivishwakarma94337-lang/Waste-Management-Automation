# Waste Management System

# Project Overview

This is a Waste Management System web application built using HTML, CSS, Bootstrap, JavaScript, Node.js, and Express.js. The system helps manage users, complaints, workers, and admin tasks related to waste management efficiently.

# Features

User Module

Register with name and location.

Submit complaints via a form.


Complaint Module

Write queries or complaints related to waste management.

Admin can view complaint status (Pending, Resolved).


Worker Module

View available workers.

Assign work to workers based on complaints.

Track worker status (Available, Busy).


Admin Module

Dashboard shows:

Number of registered users.

Number of pending complaints.

Number of works assigned.


Manage and track complaints, users, and workers.


Dashboard

Summary of users, complaints, and worker assignments.

Quick access to Contact Us module.


# Contact Us Module

Users can send messages or queries through the form.


Technologies Used

Frontend: HTML, CSS, Bootstrap, JavaScript

Backend: Node.js, Express.js

Templating: EJS



# Installation

1. Clone the repository:

git clone <https://github.com/khushivishwakarma94337-lang/Waste-Management-Automation.git>


2. Navigate to the project folder:

cd waste-management-system


3. Install dependencies:

npm install


4. Start the server:

node app.js


5. Open your browser and go to:

http://localhost:3000



# Usage

1. Users can register and submit complaints.


2. Admin can view dashboard statistics and manage users, complaints, and workers.


3. Admin can assign work to available workers and track progress.
